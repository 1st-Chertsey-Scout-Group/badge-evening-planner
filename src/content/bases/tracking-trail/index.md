One trail loops the grounds but changes language five times. Patrols set
off at intervals and bank a chalk letter at the end of every leg.

- Teach the oldest code first, crib cards out: a stick arrow means this
  way, crossed sticks not this way, a ringed stone gone home, a boxed
  arrow a message hidden a few paces on.
- Walk the five legs: a written instruction card from the gate; a voice
  note on the phone, two listens then walk; road signs clipped to posts,
  obeyed like a driver (no entry means no entry); tracking signs through
  the rough corner; and the final leg by the grounds map, kept set to the
  ground, thumb sliding along the route as it's walked.
- Bank the chalk letter at each leg's end: five letters unscramble into the
  finish word. Fastest loop with the full word wins.
- Lay-back round: each patrol lays a short trail home for the patrol
  behind, in the code it found hardest - false trails legal if signed
  honestly with crossed sticks. It only scores if the next patrol can
  actually follow it.

The whole trail stays inside the grounds; the road signs are props on
posts, never directions onto the real road.
